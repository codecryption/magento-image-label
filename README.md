# Magento Image Label

**Import product label from csv file.**  

This will update 

1. image_label
2. small_image_label
3. thumbnail_label
4. media_gallery 

CSV should be

| SKU    	| IMAGE_NAME 	|	ALT_TITLE 	|
| -------- 	| ----------- 	| -------------	|
| M-2433  	| M-2433_1.jpg  |	T-Shirt M 	|
| M-2433 	| M-2433_2.jpg  |	T-Shirt L 	|
| M-2432    | M-2432_1.jpg  |	Paint L 	|

```bash
php var/import-image-label.php
```
